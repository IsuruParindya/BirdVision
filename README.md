# BirdVision Frontend

This is the React + TypeScript frontend for the BirdVision campus research project.  
It connects to a backend API to perform bird image predictions.

---

## Overview

BirdVision is a bird recognition system developed as part of a campus research project.  
This frontend provides a user interface for uploading bird images and receiving prediction results from a machine learning model via a backend API.

---

## Features

- Upload bird images
- Preview selected images
- Send images to backend for prediction
- Display predicted bird species
- Show confidence percentage
- Loading state handling
- Error handling for failed requests
- Responsive user interface

---

## Tech Stack

- React
- TypeScript
- Vite / Create React App
- Tailwind CSS

---

## Installation

1. Clone the repository:

git clone https://github.com/your-username/birdvision-frontend.git

2. Navigate into the project directory:

cd birdvision-frontend

3. Install dependencies:

npm install

4. Start the development server:

npm run dev

The application will run at:

http://localhost:5173
